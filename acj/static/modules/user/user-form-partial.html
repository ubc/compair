Account Details

This application provides three user roles with increasing levels of permission. Students can post and edit their own answers, comments, and replies. Instructors can additionally add and edit their own courses and assignments. System Administrators can additionally create users and manage which users can access a course.

Account Login

This information will set the ComPAIR username and password for logging in without an external login (like CWL). The username you set does not appear anywehere to other users in the application but does need to be unique.
